## Usage

| Command          | Description            |
| ---------------- | ---------------------- |
| `make install`   | Install SDL2 library   |
| `make uninstall` | Uninstall SDL2 library |
| `make build`     | Compile project        |
| `make run`       | Run project            |
| `make clean`     | Clean compiled files   |
